## 🚀 How to Run

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
npm start
```

## 📝 Adding New Tools

1. Add tool definition to `/data/tools.ts`
2. Create folder: `app/tools/[tool-slug]/`
3. Add `page.tsx` with tool interface
4. Add `layout.tsx` with SEO metadata
5. Tool automatically appears everywhere!

## 🎨 Customization

- **Colors**: Edit `tailwind.config.js`
- **Categories**: Edit `/data/tools.ts`
- **Fonts**: Edit `app/layout.tsx`
- **Theme**: Edit `app/globals.css`
